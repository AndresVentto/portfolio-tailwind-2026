# 🚀 Portafolio de Andrés Vento

![Portfolio Preview](./assets/img/portafolio-img.png)

Un portafolio personal moderno y responsivo construido con tecnologías web actuales. Este proyecto refleja mi pasión por el desarrollo frontend y mi compromiso con el aprendizaje continuo en el mundo del desarrollo web.

## ✨ Características

- **Diseño Responsivo**: Optimizado para dispositivos móviles, tablets y desktop
- **Modo Oscuro/Claro**: Alternancia entre temas con persistencia en localStorage
- **Animaciones Suaves**: Transiciones fluidas y efectos visuales atractivos
- **Partículas Interactivas**: Fondo animado con Particles.js
- **Navegación Intuitiva**: Menú responsive con animaciones
- **SEO Optimizado**: Metadatos completos para motores de búsqueda y redes sociales

## 🛠️ Tecnologías Utilizadas

### Frontend
- **HTML5**: Estructura semántica y accesible
- **CSS3**: Estilos modernos con Tailwind CSS
- **JavaScript (ES6+)**: Interactividad y lógica del lado cliente

### Frameworks y Librerías
- **Tailwind CSS**: Framework CSS utility-first para diseño rápido
- **Particles.js**: Librería para efectos de partículas
- **Font Awesome & Remix Icon**: Iconografía vectorial

### Herramientas de Desarrollo
- **Node.js & npm**: Gestión de dependencias y scripts
- **Tailwind CLI**: Compilación de estilos
- **Git**: Control de versiones

## 🚀 Instalación y Uso

### Prerrequisitos
- Node.js (versión 16 o superior)
- npm o yarn

### Instalación
```bash
# Clona el repositorio
git clone https://github.com/tu-usuario/portfolio-AndresVento-2026.git

# Entra al directorio del proyecto
cd portfolio-AndresVento-2026

# Instala las dependencias
npm install
```

### Desarrollo
```bash
# Inicia el modo desarrollo con watch
npm run dev
```
Abre `public/index.html` en tu navegador o usa una extensión como Live Server.

### Producción
```bash
# Compila y copia todos los archivos a dist/
npm run build:all
```
Los archivos listos para producción estarán en la carpeta `dist/`.
